  • Markdown es una herramienta de conversin de texto a HTML para escritores web.
  • Markdown permite escribir utilizando un formato de texto plano fcil de leer y escribir, y luego convertirlo en XHTML (o HTML) estructuralmente vlido.

  • Por lo tanto, "MARKDOWN" es dos cosas:

    • 1) una sintaxis de formato de texto plano; y
    • 2) una herramienta de software, escrita en Perl, que convierte el formato de texto plano en HTML.
